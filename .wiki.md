# Project Summary
The eSignBao PHP SDK is a robust library designed to integrate electronic signature and identity verification services into applications. Targeted at both individuals and businesses, it simplifies contract management, enhances security, and streamlines digital transactions. This SDK leverages official API documentation for implementation, ensuring a reliable solution for developers seeking to incorporate electronic signatures effectively.

# Project Module Description
The project consists of the following functional modules:
1. **Authentication**:
    - `AuthService`: Manages identity verification for individuals and organizations, including personal and bank card authentication.
2. **Contract Management**:
    - `FileService`: Handles file uploads and creation based on templates.
    - `SignFlowService`: Manages the creation and oversight of signing workflows.
3. **Configuration and HTTP Client**:
    - `Configuration`: Manages application configuration settings.
    - `HttpClient`: Handles HTTP requests and responses, including error handling and signature generation.

# File Description Inventory
- **src/Client.php**: Main entry point for SDK usage, initializing authentication and contract management.
- **src/Config/Configuration.php**: Class for managing SDK configuration settings such as app ID and secret.
- **src/Http/HttpClient.php**: Class for managing HTTP requests to the eSignBao API.
- **src/Http/RetryMiddleware.php**: Implements automatic retry for HTTP requests.
- **src/Services/AuthService.php**: Class for handling personal and enterprise authentication processes.
- **src/Services/FileService.php**: Class for managing file uploads and template-based file creation.
- **src/Services/SignFlowService.php**: Class for creating and managing signing workflows.

- **src/Utils/SignatureUtil.php**: Utility functions for signature management.
- **examples/auth_example.php**: Example script demonstrating individual authentication.

- **README.md**: Documentation providing an overview and usage examples for the SDK.

# Usage
## Installation Steps
1. Install via Composer:
   ```bash
   composer require your-vendor/esignbao-sdk
   ```

## Basic Usage
```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use ESignBao\Client;

$client = new Client([
    'app_id' => 'your_app_id',
    'app_secret' => 'your_app_secret',
    'sandbox' => false,
    'max_retries' => 3,
    'enable_log' => true,
    'log_path' => '/path/to/log/esignbao.log',
]);

// Example for individual authentication
try {
    $result = $client->auth()->individualAuth('Name', 'IDCardNumber', 'Mobile');
} catch (ESignBaoException $e) {
    echo "Error: {$e->getMessage()}";
}
```
